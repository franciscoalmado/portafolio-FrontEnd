Proyecto: FrontEnd

Tecnologías: HTML5, CSS3, JavaScript vanilla, jQuery, API

Período: junio 2022

Descripción: contiene el desarrollo de un Portafolio Profesional para mostrar todos los proyectos integradores del curso. Además, se crea un juego llamado Color Game en el que el usuario deberá adivinar un color a partir de su código RGB. Se muestra un código RGB y opciones posibles para que el usuario adivine cuál es la correcta. El usuario podrá jugar en dos niveles: el fácil (con 3 colores) o el difícil (con 6). En cualquier caso, el juego le permitirá hacer click en todas las opciones hasta que dé con la correcta. También, se crea una aplicación web llamada To Do List para que el usuario pueda realizar una lista de tareas. Por último, se estructura un sitio para mostrar las condiciones meteorológicas de una ciudad, según lo especifique el usuario. Para ello, se interactua con la API del Weather Channel.
